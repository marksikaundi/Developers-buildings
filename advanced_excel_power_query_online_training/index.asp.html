Description

Microsoft Excel is a commercial spreadsheet application, written and distributed by Microsoft for Microsoft Windows and Mac OS X. At the time of developing these training modules, we have used Microsoft Excel version 2016. Microsoft Excel is a spreadsheet tool capable of performing calculations, analyzing data, and integrating information from different programs.

Audience

This training has been designed for computer users who would like to learn Microsoft Excel in easy and simple steps. It will be highly useful for those learners who do not have prior exposure to Microsoft applications.
